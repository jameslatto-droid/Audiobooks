# Free & Public-Domain Audiobook Sources

All sources below are legal. All content is public domain or CC-licensed.
Use the search tool at `custom/search-free-resources/` to query them.

## Sources

### LibriVox
- **URL**: https://librivox.org
- **API**: https://librivox.org/api/feed/audiobooks
- **License**: Public Domain (recordings are CC0)
- **Format**: MP3
- **Notes**: Human-read recordings of public-domain books. Excellent quality varies by reader.

### Internet Archive
- **URL**: https://archive.org/details/audio_bookspoetry
- **API**: https://archive.org/advancedsearch.php
- **License**: Various (Public Domain, CC BY, CC0)
- **Format**: MP3, FLAC, OGG
- **Notes**: Vast collection. Filter by `mediatype:audio` and `subject:audiobooks`.

### Project Gutenberg
- **URL**: https://www.gutenberg.org
- **Audio catalog**: https://www.gutenberg.org/browse/categories/1
- **License**: Public Domain
- **Format**: MP3 (via partner sites)
- **Notes**: Primarily text, but links to audio versions from LibriVox.

### Open Culture
- **URL**: https://www.openculture.com/freeaudiobooks
- **License**: Various (CC, Public Domain)
- **Format**: MP3, streaming
- **Notes**: Curated list — not an API. Useful for discovery.

## Usage

See `custom/search-free-resources/README.md` for how to run the search tool.

Results are saved as JSON in `custom/search-free-resources/results/`.

## Legal notes
